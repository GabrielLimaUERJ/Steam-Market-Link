Steam Market Price Checker (projeto com link)

📄 Descrição

Aplicação em Python que consulta automaticamente preços de itens do Steam Market a partir de links de listagem, permitindo análise rápida de mercado.

🛠 Tecnologias

Python

Requests

Streamlit

API pública do Steam Market

⚙ Funcionalidades

leitura automática de links do Steam Market

extração de appid e nome do item

consulta de preço atual

consulta de preço mediano

leitura de volume de vendas

tratamento de erro da API

conversão automática de moeda
